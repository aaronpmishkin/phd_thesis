# Stanford PhD Thesis
This is the repository for my PhD dissertation entitled "Convex Analysis of
Non-Convex Neural Networks". 

My dissertation was supervised by Mert Pilanci and my reading committee was
composed of Aaron Sidford, Balaji Prabhakar, Robert Tibshirani, and Stephen
Boyd.

You can find the final version online at the [Stanford University
Libraries](https://searchworks.stanford.edu/view/in00001006872).

The LaTeX template is based on
[stanford-thesis-example](https://github.com/dcroote/stanford-thesis-example).
